# 目前的每月工作时间及时薪估算
### 工作日： 
每周 一三五 二四 ，白班，每日大概5.5小时
非白班时，为错时班，每日大概2小时
### 周末：
必有一天，全天上班，每日大概8小时

故，每两周工作时长为： 
```
5.5*5 + 2*5 + 8*2 = 53.5, 即每14天中，白班5天，错时班5天，周末全日班2天。
即每两周工作约53.5小时。
```

一个月大约4.343周，故每月工作小时数为`53.5/2*4.34=116.1` 小时。

已知月工资为1800元，
故时薪约为1800/116.1 约等于 **15.5** 元。

又因法定节假日会安排值班，故时薪会更低。
